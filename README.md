# Brain-games

Brain-games is a python package for 5 different games:
* Brain-even
* Brain-calc
* Brain-gcd
* Brain-prime
* Brain-progression

# System requirements
* WSL Ubuntu
* Python 3.8

# Instructions for installation
* make load

# Instructions for run game
* Brain-even
* Brain-calc
* Brain-gcd
* Brain-prime
* Brain-progression
